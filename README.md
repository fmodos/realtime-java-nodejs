realtime-java-nodejs
====================

Palestra RealTime com Java e Node.js no TDC2014 em Florianópolis.

Detalhes dos arquivos
Real Time com Java e Node.pdf: slides da apresentação

test.js: arquivo node.js programado durante a apresentação

app.js: arquivo node.js que foi programado durante a apresentação
index.html: arquivo html que foi programado durante a apresentação

app_ok.js: arquivo node.js com os códigos OK
index_ok.html: arquivo html com os códigos OK

redis-java-cli: folder com o arquivo Java e o jar do Jedis

Para executar a demo é preciso ter o Node.js e Redis server instalado.

Step-by-step pra executar o demo
1-Instalar os pacotes utilizados no node.js
    npm install http
    npm install socket.io
    npm install redis
        
2-Iniciar o node.js na porta 3001: 
    node.exe app_ok.js 3001
3-Compilar e executar a classe java RedisTest.java
4-Abrir o browser na pagina: http://localhost:3001
5-Oi quer tc? :)
